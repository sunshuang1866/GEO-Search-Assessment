**已确认决策（2026-03-11）**：

- 四种路径都支持，skill 中可选择具体路径或全选（`paths` 参数：`forum`, `issue`, `industry`, `ai_reverse`, `all`）
- 论坛采用 API 接入方式，数据源：https://discuss.mindspore.cn/（Discourse 平台）
- 仓库 Issue 采用 GitCode API，数据源：https://gitcode.com/mindspore/mindspore/issues（需要 GITCODE_TOKEN）
- 已有 10 个问题（Q1-Q10）已完成分析
- 评分框架：两层模型（内容完整性 + 引用准确性），五种现象（A-E）
- 现象 A（官网无内容）独立为"内容缺口"层，不计入引用准确性评分
- 引用比例 = 来源级别（官方源数 / 总引用源数）
- `content_exists` 由人工预先标注（`content-labels.json`）
- 评分结果人工抽检校准 20%
- Issue 自动创建为独立 skill（issue-creator）
- Pipeline 扩展为 4 步：get-question → platform-sampler → scoring-engine → issue-creator
